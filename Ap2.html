AP2 – MediatekDocuments

Présentation du projet

Ce projet a été réalisé dans le cadre de l’épreuve AP2 du BTS SIO, option SLAM. Il consiste à développer et faire évoluer l’application MediatekDocuments, une application de gestion documentaire destinée au réseau Mediatek86.

La solution repose sur une architecture en plusieurs couches avec une application cliente développée en C# WinForms, une API REST développée en PHP et une base de données MySQL. L’application permet de gérer les documents, les commandes, les abonnements aux revues et les droits d’accès selon le service de l’utilisateur connecté.

Ce projet m’a permis de mobiliser plusieurs compétences du BTS SIO SLAM, notamment en développement d’application, gestion des données, maintenance corrective et évolutive, sécurisation des accès, utilisation d’API REST, tests et documentation technique.
